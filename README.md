# C Language Practice

This repository contains small programs and exercises created while learning and practicing the basics of the **C programming language**. The goal is to build a strong foundation by implementing simple examples and experimenting with core concepts.

## Topics Covered

- Variables and data types
- Operators
- Conditional statements (`if`, `switch`)
- Loops (`for`, `while`, `do-while`)
- Functions
- Arrays
- Pointers
- Basic memory management
- Input and output

## Project Structure

Projects are shared by difficulty. Each difficulty contains concept or exercise. Usually each file or folder focuses one task at the time.
Example format: 
```
./<difficult>/<concept>/<file>
```

Example:
/beginner/hello_world/hello_world.c

## How to Compile and Run

Make sure you have a C compiler installed (such as **gcc**).

Compile a program:

```bash
gcc program.c -o program
```

Run the program:
./program

## Purpose

This repository is intended for:
- Practicing C fundamentals
- Experimenting with small code snippets
- Building a reference for basic C concepts

## Notes

The code is mainly for learning purposes, so simplicity and clarity are prioritized over optimization.
